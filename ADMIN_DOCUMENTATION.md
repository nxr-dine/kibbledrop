## Core Admin Functions

### 1. Product Management

#### Creating New Products
1. Navigate to **Admin > Products**
2. Click **"Create New Product"** button
3. Fill in the required information:
   - **Product Name** (required)
   - **Description** (required)
   - **Price** (required)
   - **Category** (Food, Treats, Accessories, Supplements)
   - **Pet Type** (Dog, Cat, Both)
   - **Product Image** (drag & drop or click to upload)
   - **Featured Product** (toggle switch)

#### Image Upload Features
- **Supported Formats:** JPEG, PNG, WebP
- **Maximum File Size:** 5MB
- **Drag & Drop:** Simply drag images onto the upload area
- **Image Preview:** See uploaded image before saving
- **Change Images:** Click "Change Image" to replace existing photos

#### Editing Products
1. Go to **Admin > Products**
2. Click the **"Edit"** button next to any product
3. Modify any field as needed
4. Upload new images if required
5. Click **"Update Product"** to save changes

#### Managing Product Catalog
- **View All Products:** Complete list with images, prices, and categories
- **Filter & Search:** Find products quickly
- **Featured Products:** Toggle products to appear on homepage
- **Delete Products:** Remove products from catalog (use carefully)

### 2. Order Management

#### Viewing Orders
1. Navigate to **Admin > Orders**
2. See all customer orders with:
   - Order ID and date
   - Customer information
   - Order status (pending, processing, shipped, delivered)
   - Total amount
   - Items ordered

#### Order Processing
1. Click on any order to view details
2. Update order status:
   - **Pending** → **Processing** → **Shipped** → **Delivered**
3. Add tracking numbers for shipped orders
4. View customer delivery information
5. Generate invoices and order confirmations

#### Order Analytics
- Track daily, weekly, monthly sales
- Monitor order volume trends
- Identify best-selling products
- Customer order patterns

### 3. User Management

#### Customer Overview
1. Go to **Admin > Users**
2. View all registered customers:
   - Customer details (name, email, phone)
   - Registration date
   - Order history
   - Account status

#### User Administration
- **View Customer Profiles:** Complete customer information
- **Order History:** See all orders placed by each customer
- **Account Management:** Enable/disable accounts if needed
- **Customer Support:** Access customer data for support queries

### 4. Analytics & Reports

#### Dashboard Metrics
- **Total Revenue:** Real-time revenue tracking
- **Order Statistics:** Daily, weekly, monthly orders
- **Product Performance:** Best-selling items
- **Customer Growth:** New registrations and retention

#### Key Performance Indicators
- **Revenue Trends:** Track sales performance over time
- **Product Analytics:** Which products are performing best
- **Customer Metrics:** User engagement and purchasing patterns
- **Inventory Insights:** Product popularity and stock management

### 5. System Features

#### File Upload System
- **Secure Upload:** Only admin users can upload files
- **Automatic Organization:** Files stored in organized folders
- **File Validation:** Automatic checking of file types and sizes
- **Backup:** All uploads are securely stored

#### Security Features
- **Admin Authentication:** Secure login required
- **Role-Based Access:** Only admin users can access admin functions

## Troubleshooting

### Common Issues

**Cannot Upload Images:**
- Check file size (must be under 5MB)
- Ensure file format is JPEG, PNG, or WebP
- Try refreshing the page and upload again

**Order Status Not Updating:**
- Refresh the page
- Check internet connection
- Contact support if issue persists

**Cannot Access Admin Panel:**
- Verify you're using the correct login credentials
- Ensure you're accessing the correct URL
- Clear browser cache and cookies

### Getting Help

If you encounter any issues or need assistance:
1. Check this documentation first
2. Try refreshing the page or logging out and back in
3. Contact technical support with specific error messages
4. Provide screenshots when reporting issues
